
# PowerShell System Info Script

## Description
This script collects system information such as:
- Operating System Version
- Installed Software
- Disk Space Usage
- Network Adapter Details
- Get-Host(shows powershell version)


## How to Use
1. Download the `SystemInfo.ps1` script.
2. Run the script in PowerShell with the following command:
   ```powershell
   .\SystemInfo.ps1
